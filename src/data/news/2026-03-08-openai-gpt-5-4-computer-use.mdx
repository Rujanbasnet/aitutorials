OpenAI launched GPT-5.4 on March 5, positioning it as "its most capable system to date for professional use." The release is significant for one key reason: for the first time, GPT-5.4 can operate a computer on your behalf — navigating desktops, browsers, and software applications without you having to step in.

## What's New in GPT-5.4

The headline feature is **native computer use**. GPT-5.4 can click, type, switch between apps, and carry out multi-step workflows autonomously. You can ask it to pull data from a website, paste it into a spreadsheet, and format a report — all without writing code or setting up custom workflows. Previously, this kind of automation required developer setup. Now it's available directly in ChatGPT.

The model also arrives with a **1 million token context window** via the API — OpenAI's largest yet. That's roughly 750,000 words, which means you can feed entire codebases, lengthy contracts, or months of emails into a single conversation.

Accuracy has improved too. OpenAI says GPT-5.4's individual claims are 33% less likely to be false compared to GPT-5.2, and full responses are 18% less likely to contain any errors at all. For anyone who's been burned by AI confidently saying something wrong, this is a meaningful step forward.

## New Integrations

Along with the model itself, OpenAI launched:

- **ChatGPT for Excel and Google Sheets** (currently in beta), letting you use natural language to build formulas, analyse data, and create summaries inside spreadsheets
- Enterprise data integrations with **FactSet, MSCI, Third Bridge, and Moody's**, targeting financial professionals who need real-time market data inside their AI workflows

## What This Means for You

If you use ChatGPT for work, GPT-5.4 is now rolling out to **Plus, Team, and Pro subscribers**. The practical upshot: AI can now start handling multi-step computer tasks for you, not just answering questions.

For developers, the API brings the 1M context window and computer use capabilities into any application. The model consolidates what was previously split across GPT-5.3-Codex (coding) and separate reasoning models into a single, unified release.

OpenAI is clearly targeting business users — particularly the enterprise customers currently using Anthropic's Claude. Whether GPT-5.4 wins that market will depend on how reliably it handles real-world agentic tasks over the coming weeks.
