## Coding & Development

### Claude Code
**Best for:** Autonomous AI coding agent
**Price:** Usage-based (Anthropic API)
**Why we like it:** True agent-based coding. It reads, plans, and writes code across multiple files. Feels like having a senior developer on call.

### Cursor
**Best for:** AI-enhanced IDE with inline completions
**Price:** $20/month Pro
**Why we like it:** The tab completion is genuinely game-changing. Predicts your next edit with high accuracy.

### GitHub Copilot
**Best for:** Inline code suggestions in any editor
**Price:** $10/month Individual
**Why we like it:** Works in VS Code, JetBrains, Neovim. Great for boilerplate and common patterns.

### Replit Agent
**Best for:** Full-stack app generation from prompts
**Price:** Free tier + $25/month Pro
**Why we like it:** Go from idea to deployed app in minutes. Great for prototyping.

## Writing & Content

### Claude.ai
**Best for:** Long-form writing, analysis, research
**Price:** Free tier + $20/month Pro
**Why we like it:** The best writing quality of any AI. Follows instructions precisely.

### Jasper
**Best for:** Marketing copy and brand content
**Price:** From $49/month
**Why we like it:** Purpose-built for marketing teams with brand voice controls.

### Notion AI
**Best for:** Note-taking and document AI integration
**Price:** $10/month add-on
**Why we like it:** AI built right into your existing Notion workspace.

## Image Generation

### Midjourney
**Best for:** Artistic, stylized images
**Price:** From $10/month
**Why we like it:** Consistently beautiful output with distinct aesthetic quality.

### DALL-E 3
**Best for:** Accurate text-in-image and photorealistic outputs
**Price:** Included with ChatGPT Plus
**Why we like it:** Best at following detailed prompts and including text.

### Flux
**Best for:** Open-source image generation
**Price:** Free (self-hosted)
**Why we like it:** Run locally, no usage limits, great quality.

## Productivity & Automation

### Zapier AI
**Best for:** No-code automation with AI steps
**Price:** Free tier + from $20/month
**Why we like it:** Connect 6,000+ apps with AI-powered decision making.

### n8n
**Best for:** Open-source automation with LLM nodes
**Price:** Free (self-hosted) or from $20/month cloud
**Why we like it:** Full control over your automation workflows.

## Research & Analysis

### Perplexity
**Best for:** AI-powered web research
**Price:** Free tier + $20/month Pro
**Why we like it:** Answers with citations. Great for fact-checking and research.

### NotebookLM
**Best for:** Research paper analysis
**Price:** Free
**Why we like it:** Upload papers and have an AI conversation about them.
